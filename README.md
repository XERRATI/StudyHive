# Study Hive

Study Hive is a student-focused study app with a landing page, countdown, focus timer, Pomodoro controls, flashcards, Hive Coach, Garden World, vocabulary, music, terms, and privacy policy.

## Files

- `index.html` — landing page for GitHub Pages
- `study-hive-smoother.html` — main Study Hive app
- `study-hive-terms-of-service.html` — Terms of Service
- `study-hive-privacy-policy.html` — Privacy Policy
- `background-music.mp3` — backup copy of the uploaded music track, if included

## GitHub Pages setup

1. Upload all files in this folder to a GitHub repository.
2. Go to **Settings → Pages**.
3. Choose **Deploy from a branch**.
4. Select the `main` branch and `/root` folder.
5. Save.

Your website should open at:

`https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`
